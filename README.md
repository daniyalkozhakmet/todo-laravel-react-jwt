# todo-laravel-react-jwt
## Запуск проекта
### laravel
Перейти в папку backend через терминал,поставить следующий код
```
composer install
php artisan migrate
php artisan db:seed
php artisan serve
```
### react
Перейти в папку frontend через терминал,поставить следующий код
```
npm install
npm run dev
```
